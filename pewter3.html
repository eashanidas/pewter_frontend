Pewter

A platform to send money with customised messages.

What is Pewter?

Pewter is a platform which allows you to add a personalized message or video when sending money(ethers) on the Ethereum Blockchain. The main use case include attaching a gift card or a video message when gifting money to a loved one. The application will combine the user's message and money(ether) to form an ERC721 Non-Fungible Token and make it transferable . The project is intended to implement 'Paypal's gift money with designer cards' for the Ethereum Ecosystem.

What is an ERC721 Non-Fungible Token?

ERC721 is a token standard for the Ethereum Blockchain where each token is unique and non-interchangeable with other tokens. The initial purpose of the standard was to make CryptoKitties. In this project ERC721 can be used to represent the message and the associated ether as a single token. The ownership of this token can then be transferred to anyone.
